# Figures

이 폴더에는 최종 공개 보고서와 Sentaurus 결과 화면에서 추출한 실제 프로젝트 이미지를 목적별로 정리했습니다.

```text
figures/
├── overview/      # 프로젝트 첫 화면용 대표 이미지
├── preliminary/   # 선행 TCAD 실습
├── method1/       # 수치 비교 기반 최적화
├── method2/       # Ion/Ioff–SS 그래프 기반 최적화
└── code/          # 보고서에 포함된 command 화면
```

각 Markdown 문서에서는 한 이미지가 한 단계 또는 한 판단을 설명하도록 연결했습니다. 기존 `actual/` 합성 이미지와 SVG 요약 도식은 새 이미지 연결을 검증한 뒤 정리할 수 있습니다.
